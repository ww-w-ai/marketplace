# ww-w-ai Marketplace

> **ww-w-ai** — worldwide-wow AI. Wowing the world with AI.

Claude Code plugin marketplace by [ww-w-ai](https://github.com/ww-w-ai).

## Install

```bash
claude plugin marketplace add ww-w-ai/marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [cc-token-saver](https://github.com/ww-w-ai/cc-token-saver) | Claude Code keeps cutting you off? Not anymore. Spend less, code longer, and see exactly where your tokens go — zero config. |

## Install a Plugin

```bash
claude plugin install cc-token-saver@ww-w-ai
```
